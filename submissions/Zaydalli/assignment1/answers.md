**Student:** Sayidali Mahad (GitHub: Zaydalli)
**Assignment:** Section 1 & 2 - Python Foundations & Control Flow

## Project Overview
This project is a text-based Python quiz game that tests the user's knowledge using fundamental Python concepts. It tracks the user's score and provides a final result at the end of the game.

## Requirements Met
1. **Greeting & Input:** Uses the `input()` function to ask for the user's name and greets them using an f-string.
2. **Variables:** Uses an integer variable `score` initialized to `0` and incremented using the compound assignment operator (`+= 1`).
3. **Control Flow (Branching):** Utilizes `if`, `elif`, and `else` statements to evaluate whether the user's answers are correct.
4. **Multiple Conditions (Stretch Goal Achieved):** Implemented the logical `or` operator to accept multiple variations of a correct answer (e.g., `"print"` or `"print()"`).

## How to Run
To test the quiz, run the following command in the terminal:
`python quiz.py`
